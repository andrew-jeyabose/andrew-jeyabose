## 📌 Featured Projects

### 🧠 [fmri-contrastive-pretraining](https://github.com/andrew-jeyabose/fmri-contrastive-pretraining)
**Self-supervised contrastive pre-training on rs-fMRI data for foundational brain network embeddings**

SimCLR-based framework learning joint representations from 4D fMRI time-series and clinical measurements. Dual-branch spatiotemporal encoder (3D CNN spatial + 1D CNN temporal), domain-specific fMRI augmentation strategy, and NT-Xent contrastive loss. Achieves state-of-the-art on whole-brain network retrieval and pre/post-surgery classification.

`PyTorch` `Contrastive Learning` `fMRI` `Self-supervised` `Distributed Training`

📄 [Published: BioMedical Engineering OnLine 24(1):125, 2025](https://doi.org/10.1186/s12938-025-01458-6)
